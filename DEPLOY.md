# Deploy This Web App for Free

## Option 1: Vercel (Recommended - 2 minutes)

1. Go to https://vercel.com and sign up (free)
2. Click "Add New Project"
3. Import this GitHub repo OR drag this folder to Vercel
4. Click Deploy -- that's it

Your app will be live at: `https://your-project.vercel.app`

## Option 2: Netlify (2 minutes)

1. Go to https://netlify.com/drop
2. Drag the entire `ai-config-generator` folder onto the page
3. Your site is live instantly

Your app will be live at: `https://random-name.netlify.app`

## Option 3: GitHub Pages (Free)

1. Push this repo to GitHub
2. Go to Settings > Pages
3. Select `main` branch as source
4. Your site will be at: `https://username.github.io/repo-name`

## Option 4: Cloudflare Pages (Free)

1. Go to https://pages.cloudflare.com
2. Connect your GitHub repo
3. Build settings: leave default (it's static HTML)
4. Deploy

## After Deploying

1. Update the Gumroad link in `index.html` (search for "gumroad.com" and replace with your actual Gumroad product URL)
2. Test the config generator works
3. Share the URL everywhere
